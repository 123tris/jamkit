# JamKit Documentation

Rapid game-jam toolkit for Unity 6. Start with the [README](../README.md) for architecture and the component catalog; these pages are the hands-on guides:

- **[Hour Zero Checklist](hour-zero-checklist.md)** — the first ten minutes of a jam, in order.
- **[Pong in 60 Seconds](pong-in-60-seconds.md)** — two paddles, a ball, goals, two players, done.
- **[Frogger in 5 Minutes](frogger-in-5-minutes.md)** — grid movement, traffic, kill water, goal.
- **[Graduating to Feel](graduating-to-feel.md)** — swapping Juice Lite for MoreMountains Feel without rewiring.

These walkthroughs double as manual test scripts: if a step doesn't work as written, that's a JamKit bug — file it.
